# BIBISHKA Admin Bot

Telegram-бот на Python, aiogram 3 и SQLite для чата Бибишки: FAQ, модерация, админ-панель, реклама, награды, состав админов и локальный бесплатный AI-помощник.

## Возможности

- `/start` с разделами: FAQ, Соцсети, Стримы, Реклама, Состав, Награды.
- FAQ с красивыми кнопками и разделами с большой буквы.
- Автоответы по ключевым словам.
- Локальный AI без внешних API: напишите `Бибишка, вопрос`.
- Реклама: заявки уходят на ID `8436225978` в личный чат с ботом.
- Админ-панель `/admin` для FAQ, админов, статистики и настроек.
- Модерация как в админ-ботах: варны, муты, временные баны, кик.
- Фильтры мата, спама, ссылок и капса.
- Награды: `/award`, `/awards`, `/unaward`.
- Состав админов: `/staff`.
- Автоматическое создание SQLite-базы при первом запуске.

## 1. Установка зависимостей

```bash
python -m venv .venv
```

Windows PowerShell:

```bash
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

Linux/macOS:

```bash
source .venv/bin/activate
pip install -r requirements.txt
```

## 2. Настройка `.env`

Скопируйте пример:

```bash
copy .env.example .env
```

Для Linux/macOS:

```bash
cp .env.example .env
```

Заполните `.env`:

```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id
DATABASE_PATH=data/bot.db
```

Несколько админов можно указать через запятую:

```env
ADMIN_IDS=111111111,222222222
```

## 3. Запуск бота

```bash
python bot.py
```

При первом запуске бот создаст `data/bot.db`, таблицы, стартовые FAQ и настройки.

## 4. Назначение первого администратора

1. Узнайте свой Telegram ID, например через `@userinfobot`.
2. Добавьте ID в `.env`:

```env
ADMIN_IDS=123456789
```

3. Перезапустите бота.
4. Напишите `/admin`.

Новых админов можно добавлять через:

`/admin` → `👥 Управление администраторами` → `➕ Добавить ID администратора`.

## Модерация

Команды работают ответом на сообщение или через ID пользователя:

```text
/warn причина
/unwarn
/mute 1m причина
/mute 1h причина
/mute 1d причина
/unmute
/ban 1m причина
/ban 1h причина
/ban 1d причина
/ban причина
/unban USER_ID
/kick причина
```

Если пользователь получает лимит варнов, например `3/3`, бот выдает мут на время из настроек и сбрасывает варны. Настраивается в `/admin` → `⚙️ Настройки`.

## Реклама

Пользователь открывает `/start` → `📣 Реклама` или пишет `/ads`, затем отправляет текст, фото, видео или файл. Бот копирует заявку на ID `8436225978`.

ID получателя можно поменять:

`/admin` → `⚙️ Настройки` → `📣 ID рекламы`.

Важно: получатель рекламы должен сначала открыть чат с ботом и нажать `/start`, иначе Telegram не даст боту написать ему первым.

## AI-помощник

AI работает локально и бесплатно, без API-ключей. Формат:

```text
Бибишка, сколько лет Бибисоре?
Бибишка, где она живет?
Бибишка, кто ее лучшая подруга?
```

В базу знаний добавлены факты: Бибисоре 15 лет, она в 9 классе, родилась 25 августа, живет в Узбекистане в Андижане, лучшая подруга — Садокат, друзья — Нурик, Абубакр и Эмиль.
